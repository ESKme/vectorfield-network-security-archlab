# Code of Conduct

This project follows a code of conduct to ensure a respectful,
professional, and technically focused collaboration environment.

---

## Purpose

The VECTORFIELD repository is part of the **ESKme learning and research projects**  
and aims to provide open educational material in the field of **enterprise
network security architecture and cybersecurity methodology**.

This repository encourages professional discussion, constructive critique,
and collaborative learning within the field of cybersecurity architecture.

To support this goal, participants are expected to maintain a respectful,
inclusive, and technically focused environment.

---

## Expected Behavior

Participants in this repository are expected to:

- engage in **respectful and professional communication**
- provide **constructive feedback and well-reasoned critique**
- support a **learning-oriented environment**
- remain focused on **technical and methodological discussion**
- respect the work and contributions of others

Disagreements about architectural decisions or methodologies are welcome
when expressed in a constructive and professional manner.

---

## Unacceptable Behavior

The following behavior is not acceptable in this repository:

- harassment or discriminatory language
- personal attacks or hostile communication
- trolling, deliberate disruption, or inflammatory comments
- posting irrelevant promotional or spam content
- misrepresenting the work or contributions of others

Discussions should remain focused on **cybersecurity architecture,
methodology, and technical reasoning**.

---

## Moderation

The repository maintainer reserves the right to:

- remove comments or contributions that violate this code of conduct
- close discussions that become unproductive or hostile
- restrict participation if necessary to maintain a constructive environment

Moderation decisions aim to protect the **professional and educational
character** of the project.

---

## Reporting Concerns

If you experience or observe behavior that violates this Code of Conduct,
you may report it to the maintainer:

Email: **contact@ESKme.net**

Reports will be reviewed and handled with discretion.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- repository discussions
- other interactions related to this project
