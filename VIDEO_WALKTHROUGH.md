# Video Walkthrough

This video provides a guided overview of the 
**VECTORFIELD Enterprise Network Security Architecture Study**.

The presentation explains the architecture, security model, 
design decisions, and validation approach used in the project.

## Topics Covered

The walkthrough covers the following aspects of the architecture:

- project background and architectural goals
- zone-based network segmentation
- firewall enforcement and default-deny model
- security monitoring and SOC-readiness
- validation methodology

---

## Purpose of the Video

The video provides a concise walkthrough of the architecture 
for readers who prefer a visual explanation of the design.

The complete technical documentation is available in the **docs/** directory, 
and the full study can be found in the **pdfs/** directory.
